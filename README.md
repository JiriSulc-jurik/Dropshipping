Při tvorbě webu jsem použil AI nástroj ChatGPT (OpenAI, GPT-5.1).

Použité prompty:
- „Chci web o dropshippingu, který splní školní zadání.“
- „Uprav vzhled, aby byl lepší a přehlednější.“
- „Začni projekt od začátku, rozděl kód do souborů.“
- „Udělej kompletní redesign se svítícími barvami.“
- „Napiš stručnou dokumentaci postupu a technologií.“

Stručný pracovní postup:
1. Založil jsem repozitář na GitHubu a projekt ve Visual Studiu / VS Code.
2. Pomocí AI jsem vytvořil texty, strukturu webu a rozložení stránek.
3. AI vygenerovala HTML, CSS a JavaScript.
4. Provedl jsem několik úprav designu podle požadavků.
5. Výsledek jsem nahrál na GitHub a doplnil README.md.


Použité technologie

- HTML5 – struktura stránek a obsah.
- CSS3 – rozvržení, responzivita, neonový (svítící) design.
- JavaScript (ES6) – formulář a interaktivní prvky.
- Git a GitHub – verzování a sdílení projektu.
- Visual Studio / VS Code – úprava kódu.
